About Minimum Spanning Tree

A Minimum Spanning Tree (MST) connects all vertices of a weighted graph using the smallest possible total edge weight without forming cycles.


Prim’s Algorithm:
  • Starts from a vertex
  • Adds smallest connecting edge
  • Expands the tree

Kruskal’s Algorithm:
  • Sorts edges by weight
  • Adds smallest edges first
  • Avoids cycles using Union-Find
